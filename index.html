Nossas linhas de produtos

Linha para Lavanderia

Utilizada para desinfecção, roupas brancas, limpeza em geral.

Linha para Casa

Indicado para limpeza de cozinhas, banheiros, é desengordurante eficiente para pias, bancadas, fogões, mesas etc.

Linha para Automotivo

Limpa dá brilho e protege seu veículo deixando um suave perfume no interior do carro.

Linha de Detergente para as Mãos

Destinado à limpeza de mãos, possui uma fórmula balanceada que proporciona um alto rendimento.

Linha para Limpeza Pesada

Alto poder bactericida, usado na limpeza de pisos, lajotas, pedras, azulejos, paredes, pode ser utilizado também diluído.

Linha de Aromatizante de Ambientes

Destinado para aromatizar ambientes como salas, quartos, banheiros, cozinhas e ambientes fechados.
